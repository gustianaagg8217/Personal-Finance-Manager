🎯 QUICK START (LANGSUNG KERJA)

Langkah 1 - Setup Token (Pilih SATU):

✅ OPSI A - Tercepat (One-Click):
   double-click: quick_setup_telegram.bat
   
✅ OPSI B - GUI (Paling Visual):
   Run: python launcher.py
   Click: 🤖 Setup Telegram
   
✅ OPSI C - Menu (Tradisional):
   Run: launcher.bat
   Select: 2) Start Telegram Bot
   Follow prompts

Langkah 2 - Masukkan Token:
   1. Buka Telegram → Cari: @BotFather
   2. Kirim: /newbot
   3. Ikuti instruksi (kasih nama bot, dll)
   4. Copy token (format: 1234567890:ABC...)
   5. Paste ke setup

Langkah 3 - Restart Terminal:
   Tutup terminal, buka yang baru

Langkah 4 - Test:
   launcher.bat → Select 2 → Bot should start!
