❌ ERROR RECOVERY
════════════════════════════════════════════════════════════════════════════════

If bot doesn't start:
    1. Check token: echo %TELEGRAM_BOT_TOKEN%
       ↓
    2. If empty:
       Run: quick_setup_telegram.bat
       ↓
    3. If not empty:
       Run: setup_telegram.bat → Select 3) Test Token
       ↓
    4. If test fails:
       - Token invalid? Reset in @BotFather
       - Network issue? Check internet
       - Bot inactive? Activate in BotFather
       ↓
    5. Still not working?
       Run manually: python run_telegram_bot.py
       → See detailed error output
       → Check: telegram_bot.log
       → Read: TELEGRAM_SETUP_GUIDE.md
